关于@media

	列子:
			<style>
				div{
					color: red;
				}
				@media screen(max-width: 900px){
					div{
						color: green;
					}
				}
			</style>
			<div>
				test
			</div>

定义:
窗口大小不同，样式改变
注意事项:
1，这个要写在原有的css下面
2，和原有的css旋转器保持一致 比如 xxx>xxx x>w
一定要写一模一样
3,并且当max-width的时候 ，大的数值要写在上面
小的写在下面， 参考本页面。
