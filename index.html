关于@media

	列子:
			<style>
				div{
					color: red;
				}
				@media and screen(max-width: 900px){
					div{
						color: green;
					}
				}
			</style>
			<div>
				test
			</div>

定义:
窗口大小不同，样式改变
注意事项:
1，这个要写在原有的css下面
2，和原有的css旋转器保持一致 比如 xxx>xxx x>w
一定要写一模一样
3,并且当max-width的时候 ，大的数值要写在上面，用min-width的时候 小的写在上面
小的写在下面， 参考本页面。

jQuery键盘事件

1、keydown() keydown 事件会在键盘按下时触发。
2、keypress() keypress 事件会在敲击按键时触发，我们可以理解为按下并抬起同一个按键。
3、keyup() keyup 事件会在按键释放时触发，也就是你按下键盘起来后的事件。
二、获得键盘上对应的 ascII 码：$(document).keydown(function (event) { alert(event.keyCode); });
